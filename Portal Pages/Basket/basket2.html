Basket API

When the Baskets Service is used in a Customer-to-Business mode all Baskets are created as implicitly intended for submission as a single Order to be processed in whole. In this context a single Basket Order Details Item is created in the Basket when the Basket is created. All Operations of the Basket Orders Service serve to maintain details stored in that Item. Some of the Operations of this service use Data Structures in Requests and Responses that are specific to the Service API. These are described in section Basket Orders API Data Structures of Chapter 15, The Basket Data Structure.

Delivery Options

This Operation Lists the available delivery options for a specified delivery address. Each of the delivery options listed indicates what products (including quantity) the delivery options can fulfill.
